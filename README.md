# clang-sanitizer-sample

Clang sanitizer sample programs that confirm the sanitizer detects what kind of bugs.

## Sanitizers

- AddressSanitizer(ASan)
- MemorySanitizer(MSan)
- ThreadSanitizer(TSan)
- UBSanitizer(UBSan)
